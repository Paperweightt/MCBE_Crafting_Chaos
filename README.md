## build instructions

#### setup

```powershell
  npm i
  Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
```

#### build

```powershell
  npm run generate
  npm run local-deploy
```

#### auto build

```powershell
  npm run local-deploy -- --watch
```

currently manual
